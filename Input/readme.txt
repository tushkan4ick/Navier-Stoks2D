tp_id - Тип граничного условия
1 - стенка
2 - сверхзвуковой вход
3 - ось симметрии
4 - сверхзвуковой выход
5 - дозвуковой вход
6 - дозвуковой выход
*************************************************************************************
stp_id - подтип граничного условия
(стенка)
1 - температура стенки
2 - тепловой потока
3 - конвекция
(дозвуковой вход)
1 - компоненты скорости, давление и температура, далее давление определяется при расчете
2 - полное давление, давление, угол атаки (в РАДИАНАХ) и температура, далее скорость определяется при расчете
3 - массовый расход, давление и температура, далее давление определяется при расчете
(дозвуковой выход)
скорость по умолчанию только х-компонента
*************************************************************************************
bc - id границы, по которой идет инициализация
